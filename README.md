# Sales Forecasting & Predictive Analytics Dashboard

## 📌 Project Overview

This project focuses on analyzing historical sales data and forecasting future sales trends using Power BI. The dashboard provides interactive visualizations, key performance indicators (KPIs), and predictive insights to support data-driven business decision-making.

The project demonstrates the use of business intelligence and predictive analytics techniques to transform raw sales data into meaningful insights.

---

## 🎯 Objective

* Analyze historical sales performance.
* Identify sales trends and growth patterns.
* Forecast future sales using Power BI forecasting capabilities.
* Visualize key business metrics through an interactive dashboard.
* Support strategic planning and decision-making.

---

## 🛠️ Tools & Technologies

* Power BI Desktop
* Power Query
* DAX (Data Analysis Expressions)
* Microsoft Excel
* Forecast Analytics (Power BI)

---

## 📂 Dataset Information

The dataset contains monthly sales records from January 2022 to December 2024.

### Dataset Fields

| Column Name | Description            |
| ----------- | ---------------------- |
| Date        | Sales transaction date |
| Product     | Product name           |
| Region      | Sales region           |
| Sales       | Sales amount           |

---

## 🔄 Data Preparation

The following preprocessing steps were performed:

* Imported Excel dataset into Power BI.
* Verified data types.
* Removed inconsistencies and cleaned data.
* Formatted date fields.
* Created measures using DAX.

---

## 📊 Dashboard Features

### 1. Total Sales KPI

Displays overall sales performance.

### 2. Average Sales KPI

Shows average sales value across the dataset.

### 3. Growth Percentage KPI

Measures business growth rate.

### 4. Monthly Sales Trend

Visualizes sales performance over time.

### 5. Annual Sales Trend

Compares yearly sales growth.

### 6. Sales Performance Against Target

Gauge chart comparing actual sales with target sales.

### 7. Future Sales Forecast

Predicts future sales trends from 2025 to 2030 using Power BI forecasting.

### 8. Interactive Filters

* Region Filter
* Product Filter

### 9. Data Table

Displays detailed sales records.

## 🚀 Forecasting Approach

Power BI's built-in forecasting functionality was applied to historical sales data.

### Forecast Settings

* Forecast Length: 6 Years
* Confidence Interval: 95%
* Seasonality: Auto Detection

The model predicts future sales performance based on historical patterns and trends.
